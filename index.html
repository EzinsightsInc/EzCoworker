EzCoworker

Your AI Agents. Your Infrastructure. Your Rules.

Self-hosted AI agent platform powered by Claude Code CLI. Connect any LLM — local or cloud. Run agents in isolated Docker containers. Process files, write code, generate documents. 100% on your servers.

Isolated Agent Containers

Every agent run spawns a sandboxed Docker container with per-user workspace volumes. Full filesystem access within the container. Zero cross-contamination.

Docker Isolated
🔧

MCP Tool Integration

Plug in any Model Context Protocol server as agent tooling. SSE and stdio transports supported. Bundle tools + instructions as reusable plugins.

MCP Native

File Workspace

Upload files to the agent's input folder. Agent reads, processes, and writes output. Download results via signed JWT links. Supports CSV, PDF, Excel, ZIP, images.

Get Running in Minutes

Quick Start Guide

From zero to running AI agents on your own machine in under 10 minutes.

01 📦

Clone the Repo

Clone EzCoworker Community from GitHub.

git clone github.com/ezinsights-ai/ezcoworker
02 🐳

Build Agent Image

Build the Claude Code agent image with Python, LibreOffice, and all document tools baked in.

docker build -f agent.Dockerfile -t claude-agent-image-community .
03 ⚙️

Configure .env

Copy the example env file and add your API keys for any LLM providers you want to use.

cp .env.example .env
04 🚀

Launch Platform

Start all services with one command. Frontend, backend, and database come up together.

docker-compose up -d
05 👤

Create Admin

Create the initial admin account via the setup endpoint using your ADMIN_SETUP_KEY.

POST /api/admin/setup
06 🤖

Start Chatting

Open the platform in your browser, register a user, choose a model, and start your first agent session.

http://localhost:3600
